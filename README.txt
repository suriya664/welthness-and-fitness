==================================
CUSTOMIZATION
==================================

1. CHANGE COLORS:
   Edit: assets/css/style.css
   Find: :root { --primary-color: #4CAF50; }
   Change to your brand color

2. UPDATE LOGO:
   Edit: partials/header.html
   Replace: Ayura<span>Well</span>
   With: Your<span>Brand</span>

3. MODIFY CONTENT:
   Edit any page in pages/ folder
   Update text, images, and links

4. SETUP FORMS:
   Edit: assets/js/ajax.js
   Add your API endpoints
